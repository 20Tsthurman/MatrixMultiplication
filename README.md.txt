# MatrixMultiplication

MatrixMultiplication is a JavaFX application that demonstrates the performance difference between single-threaded and multi-threaded matrix multiplication.

## Features

- Perform matrix multiplication using a single core or multiple cores.
- Compare the time taken for each operation.
- Visualize performance data through an integrated chart.

## Prerequisites

Before you begin, ensure you have met the following requirements:
- You have installed Java JDK 11 or above.
- You have installed Maven.
